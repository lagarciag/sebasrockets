## Development

### Prerequisites

- Python 3.10+

### Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Local Preview

```bash
mkdocs serve
```

### Deploy

```bash
mkdocs gh-deploy
```
